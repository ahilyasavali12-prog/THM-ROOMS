## 📋 Overview

This room explores the many career paths available in cybersecurity — from offensive roles like penetration testing to defensive roles in SOC analysis. It helps you understand which direction to go and what skills each path requires.

---

## 🔴 Offensive Security Roles (Red Team)

These roles involve **attacking** systems to find vulnerabilities before real attackers do.

| Role | Responsibilities |
|------|-----------------|
| **Penetration Tester** | Simulate attacks against systems/apps to find vulnerabilities |
| **Red Team Operator** | Conduct full adversarial simulations against organisations |
| **Exploit Developer** | Write custom exploits and tools |
| **Bug Bounty Hunter** | Find vulnerabilities in public programs for rewards |

---

## 🔵 Defensive Security Roles (Blue Team)

These roles **protect** organisations from attacks.

| Role | Responsibilities |
|------|-----------------|
| **SOC Analyst** | Monitor SIEM alerts, investigate incidents |
| **Incident Responder** | Contain and recover from active attacks |
| **Threat Intelligence Analyst** | Research threat actors and TTPs |
| **Digital Forensics Analyst** | Investigate breaches, collect evidence |
| **Malware Analyst** | Reverse engineer malicious software |

---

## 🟣 Other Key Roles

| Role | Description |
|------|-------------|
| **Security Engineer** | Design and implement security controls |
| **Cloud Security Architect** | Secure cloud infrastructure |
| **GRC Analyst** | Governance, Risk, and Compliance |
| **Security Awareness Trainer** | Educate staff on security practices |

---

## 📈 Career Pathways

```
Entry Level
  ├── SOC Analyst (L1) → SOC Analyst (L2/L3) → Incident Responder
  ├── IT Support → Network Admin → Security Engineer
  └── Junior Pentester → Senior Pentester → Red Team Lead

Certifications Roadmap:
  CompTIA A+ → Network+ → Security+ → CEH / OSCP → CISSP
```

## 📝 Key Takeaways

- Cybersecurity has diverse roles — you don't have to be a hacker to work in security.
- Both offensive and defensive skills are valuable — many professionals develop expertise in both.
- Practical labs (like TryHackMe) are as important as certifications for demonstrating skill.
- The field has a significant skills shortage — it's an excellent career choice.
